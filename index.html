About me

I'm a software engineer with a mobile-first focus and 4+ years building production apps, dashboards and backend services. I bounce between iOS, web and backend — jack of all trades, maybe, but I try to do each one well.

At Karage I built and shipped two App Store apps: My Karage, a customer-facing iOS app with booking and payments that reached 5,000+ users, and Karage Kash, an iPadOS point-of-sale app in Swift & SwiftUI. Alongside the apps I work on a Next.js/TypeScript reporting dashboard and ASP.NET Core microservices — so I've seen products from the UI down to the database.

I'm not a fan of quick fixes that break in a month. I write code for the person who maintains it later (usually me), I genuinely enjoy code reviews, and I lean on AI tools like Claude Code daily to move faster without shipping sloppy work. Always learning, always building, always shipping.
